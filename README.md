# Civil-Materials-Calc.
Projeto acadêmico de cálculo de concreto estrutural — interface gráfica para cálculos de concreto armado.

# 🏗️ Calculadora de Concreto Armado

Este projeto é uma aplicação em **Python + Flet** para calcular o volume de concreto e os materiais necessários (cimento, areia e brita) em diferentes tipos de elementos estruturais.

# 🚀 Funcionalidades

- Cálculo de volume a partir das dimensões (largura, altura, comprimento).
- Seleção do tipo de elemento: viga/pilar, baldrame, laje com Fck 25 ou laje com Fck 30.
- Exibição dos materiais necessários com base no volume calculado.
- Interface gráfica bem  simples e intuitiva via navegador.
